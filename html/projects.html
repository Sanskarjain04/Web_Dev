Trip Planner


A Website to plan trips for the people who wants to go on a trip.


How it works:-

We used GPT3 to make this project, we used the GPT3 API key and connected our website with GPT3 Playgroung.
The website asks three thing:

1.The starting day of the trip.
2.The end day of the trip.
3.The Place where you want to go.

After giving these three information the website sends the information to GPT3, it receives the information, generates a plan and sends it back to the website and the website shows the plan to the user on the screen.
